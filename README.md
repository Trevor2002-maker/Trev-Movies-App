## Installation

Install trev-movies-app with npm

```bash
  npm install trev-movies-app
  cd trev-movies-app
```

## Run Locally

Clone the project

```bash
  git clone https://github.com/Trevor2002-maker/Trev-Movies-App.git
```

Go to the project directory

```bash
  cd Trev-Movies-App
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
